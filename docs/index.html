Why Ring

烦人的节假日闹钟

让闹钟知道今天该不该响。面向中国大陆节假日、补班日、大小周、轮班等场景的本地智能闹钟工具。

核心特性

  • 隐私优先：数据保存在设备本地，不上传服务器
  • 无广告：纯净体验，无广告打扰
  • 无账号：无需注册，开箱即用
  • 本地优先：不依赖网络，离线可用
  • Apple Watch：支持 Apple Watch同步响铃提醒

适用场景

  • 春节、国庆等法定节假日调休
  • 周末补班日特殊闹钟
  • 双休 / 单休 / 大小周
  • 上四休三、上四休四等轮班
  • 完全自定义的排班规则

核心概念

规则方案 一组规则组的组合，可为不同生活节奏创建不同方案（双休、单休、大小周等）。
规则组 用于判断某一天属于什么类型：工作日、补班日、休息日、法定节假日。
闹钟分组 决定某组闹钟在哪类日期响：工作日、休息日或每天。
同步到系统 变更先保存在本地，确认预览后通过 AlarmKit 写入 iOS 系统闹钟。

首次使用

  1. 打开 App，阅读欢迎说明、用户协议和隐私协议。
  2. 进入首页，查看当前日历方案和临近闹钟。
  3. 前往「闹钟管理」，新建分组并添加闹钟。
  4. 点击「同步到系统」，选择同步范围（建议 90 天）。
  5. 查看同步预览，确认后写入系统闹钟。
  6. 根据系统提示授权 AlarmKit。
